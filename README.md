# 🎮 Epic Whack-a-Mole Game

A fun and interactive **Whack-a-Mole browser game** built using HTML, CSS, and JavaScript.  
Test your reflexes, beat the timer, and aim for the highest score! 🚀

---



## 🎯 Features

- 🎮 Classic Whack-a-Mole gameplay
- ⏱️ 30-second timer challenge
- 🏆 Real-time scoring system
- 💥 Combo bonus system
- ✨ Animated UI (stars + effects)
- 📱 Responsive design (mobile friendly)
- 🎨 Modern glassmorphism UI

---

## 🛠️ Built With

- HTML5
- CSS3 (Animations + Flex/Grid)
- JavaScript (Game logic)
